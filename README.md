# Net-Sec-Challenge-Notes
Apply practical network security skills including scanning, enumeration, and service analysis in a hands-on environment.
⚙️ Key Concepts Practiced
Port Scanning
Identifying open ports using tools like Nmap
Detecting services running on non-standard ports
Service Enumeration
Gathering detailed information about services (FTP, HTTP, etc.)
Version detection using banners and scripts
Protocol Interaction
HTTP → Using Telnet / manual requests
FTP → Accessing files, checking configurations
POP3/SMTP → Understanding email services
Banner Grabbing
Extracting service/version info from server responses
Hidden Information Discovery
Finding flags in HTTP headers, files, and directories
🛠️ Tools Used
Nmap → Port scanning, service/version detection
Telnet / Netcat → Manual interaction with services
FTP Client → File access and enumeration
Browser / Curl → HTTP requests
🧠 Key Learnings
Not all services run on default ports → always scan full range
Enumeration is critical after discovering open ports
Manual interaction helps understand protocols deeply
Small misconfigurations can expose sensitive data
🚀 Outcome
Improved practical understanding of network services
Strengthened enumeration and analysis skills
Gained confidence in solving real-world security challenges
